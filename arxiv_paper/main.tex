\begin{abstract}
Large Language Models (LLMs) equipped with standard Retrieval-Augmented Generation (RAG) suffer from structural blindness: they cannot distinguish between role-swapped propositions due to the commutative nature of continuous vector search. We introduce a composition-episodic cognitive memory architecture based on Vector Symbolic Architectures (VSA). By decoupling semantic embedding from structural role-binding, the system preserves exact structural relationships and sequential episodic order without sacrificing continuous generalization. We demonstrate that the architecture sustains graceful capacity scaling up to 16,000 facts, achieves near-perfect episodic order recall within a bounded working window, and maintains high semantic fidelity between the original dense embedding space and the high-dimensional hyperdimensional space. Our results show that this hybrid neuro-symbolic approach significantly outperforms standard RAG baselines on role-ambiguous queries while providing a mathematically rigorous foundation for verifiable AI memory.
\end{abstract}

\section{Introduction}
\label{sec:intro}

The integration of external memory into Large Language Models (LLMs) via Retrieval-Augmented Generation (RAG) \cite{lewis2020rag} has become the standard method to overcome token limits and provide agents with external knowledge. However, flat continuous vector search---the engine powering standard RAG---is inherently blind to structure. Because sentence embeddings rely on summing or averaging attention over token distributions, propositions that share identical lexemes but differ in relational structure often collapse into identical dense neighborhoods. For example, "Alice killed Bob" and "Bob killed Alice" produce near-identical cosine similarities, leading LLM agents to hallucinate roles when queried \cite{kanerva2009hyperdimensional}.

We propose \textbf{Astrum Verum}, a cognitive memory architecture that solves structural blindness by applying Vector Symbolic Architectures (VSA) \cite{gayler2004vsa, kanerva2009hyperdimensional} on top of modern transformer embeddings. Rather than using the LLM as an implicit knowledge base, we utilize it strictly as a linguistic parser to extract semantic components (subject, relation, object). These components are grounded into $10,000$-dimensional bipolar hypervectors using SimHash projection, and then algebraically bound and bundled using Multiply-Add-Permute (MAP) operations.

Our approach revives Sparse Distributed Memory (SDM) principles \cite{kanerva1988sdm} in a modern context. By separating semantic grounding from structural encoding, we enable \textit{associative} recall (by meaning) alongside \textit{structural} recall (by role), creating a memory that reasons over symbols rather than just retrieving text chunks.

\section{Related Work}
\label{sec:related}

\textbf{Vector Symbolic Architectures (VSA)} and \textbf{Holographic Reduced Representations (HRR)} \cite{plate1995holographic} provide a mathematical framework for representing compositional structure in distributed representations, extending early work on Tensor Product Variable Binding by Smolensky \cite{smolensky1990tensor}. Kanerva's Sparse Distributed Memory (SDM) \cite{kanerva1988sdm} and subsequent work on Hyperdimensional Computing (HDC) \cite{kanerva2009hyperdimensional} established the theoretical foundation for high-dimensional associative memory. The field has matured through operator algebras like Multiply-Add-Permute (MAP) by Gayler \cite{gayler2004vsa} and structural representations by Rachkovskij \cite{rachkovskij2001representation}.

The theoretical efficiency of these high-dimensional spaces is deeply connected to the mathematics of sphere packing in 8 and 24 dimensions, as formally proven by Viazovska \cite{viazovska2017sphere} and Cohn et al. \cite{cohn2017sphere}, which provides a geometric justification for the orthogonality of random vectors in $D=10,000$.

Recent neuro-vector-symbolic architectures have successfully bridged neural networks with VSA for reasoning tasks \cite{hersche2023neuro, kleyko2022vector}. Simultaneously, the deep learning community has demonstrated that modern attention mechanisms \cite{vaswani2017attention} can be mathematically understood as approximations of SDM \cite{bricken2021attention}, and that Hopfield Networks function similarly to transformer attention \cite{ramsauer2020hopfield}. Practical software implementations, such as Torchhd \cite{heddes2022torchhd} and Meier's VSA in Clojure \cite{meier2022vsa}, have further democratized access to these paradigms. Our work establishes a specific bridge between these theories and real-world LLM agent memory.

\subsection{SimHash Grounding and Semantic Fidelity}

\subsection{SimHash Grounding and Semantic Fidelity}
To bridge the gap between continuous dense embeddings (e.g., from Sentence-Transformers) and bipolar VSA hypervectors, we use SimHash projection \cite{charikar2002similarity}. A dense vector $v \in \mathbb{R}^d$ is multiplied by a random projection matrix $P \in \mathbb{R}^{d \times D}$. The projection matrix entries are i.i.d. Gaussian $P_{ij} \sim \mathcal{N}(0,1)$. The resulting hypervector is:
\begin{equation}
    H = \mathrm{sgn}(v P), \quad H \in \{-1, 1\}^D
\end{equation}
This ensures that semantically similar tokens remain geometrically proximate in the high-dimensional space, effectively mapping continuous semantic neighborhoods onto an orthogonal boolean hypercube.

\subsection{Vector Symbolic Architecture (VSA)}
We employ bipolar hypervectors $\{-1, 1\}^D$ ($D=10,000$) and the MAP algebra:
\begin{itemize}
    \item \textbf{Binding ($\otimes$):} Element-wise multiplication (XOR in binary) is used to bind roles to fillers (e.g., $Role_{subj} \otimes Filler_{Alice}$).
    \item \textbf{Bundling ($\oplus$):} Element-wise addition followed by a majority sign operation $\text{sgn}()$ combines multiple elements.
    \item \textbf{Permutation ($\Pi$):} Cyclic shifts encode episodic sequences, mapping temporal order into the geometric space.
\end{itemize}

A fact is encoded as a bundled hypervector of bound roles, as illustrated in the architecture overview (Figure \ref{fig:arch}):
\begin{equation}
    Fact = \mathrm{sgn}(R_{subj} \otimes S + R_{rel} \otimes R + R_{obj} \otimes O)
\end{equation}

\subsection{Item Memory and Match Quality}
A known challenge in VSA is capacity collapse: bundling thousands of facts into a single global superposition vector rapidly causes the noise to overwhelm the signal. To circumvent this, each fact is stored as an independent hypervector in an \textit{Item Memory}. Retrieval is performed via cosine similarity between a structural probe and all stored fact vectors. This decouples system capacity from VSA superposition limits.

However, searching across an Item Memory introduces the problem of \textit{Decoded Clarity Hallucinations}. When unbinding a retrieved fact to find an answer, the expected cosine similarity for a true signal is $0.5$ ($50\sigma$ above the noise floor of $\sigma=0.01$). But across a large dataset, a non-relevant fact might randomly produce a noisy, unbound vector that happens to align closely with a dictionary concept (e.g., score = $0.502$), outscoring the true fact (score = $0.493$).

To prevent these decoding hallucinations, we define \textbf{Match Quality} ($M$) as the product of global relevance and local clarity. Let $concept^*$ be the candidate from the codebook $\mathcal{C}$ that maximizes decoding clarity:
\begin{equation}
    concept^* = \text{argmax}_{c \in \mathcal{C}} \text{cosine}(\text{unbound}, c)
\end{equation}
The final score is defined as:
\begin{equation}
    M = \text{cosine}(\text{probe}, \text{fact}) \times \text{cosine}(\text{unbound}, concept^*)
\end{equation}
Retrieval succeeds only if the top candidate exceeds dual thresholds: $\tau_1 = 0.2$ for fact relevance and $\tau_2 = 0.35$ for decode clarity. As shown in Figure \ref{fig:results}, this metric maintains a clear separation between positive matches and noise. This formulation enables an \textbf{Early Exit} strategy: if the first retrieved candidate passes the thresholds, search terminates immediately, yielding a roughly $5\times$ speedup during retrieval (achieving $\sim 0.15$s per query on a single CPU core at $N=16,000$).

\subsection{Bounded Working Memory}
Episodic sequences are stored by applying a cyclic permutation to the cumulative bundle for each new item. To prevent vector saturation, we maintain a sliding window of length $W = 150$: only the last $W$ items are kept in the active episodic vector. Older items are transferred to the long-term Item Memory and are not used for order recall, effectively simulating the transition from working to long-term memory.

At $N=1,000$ facts, RAG achieved 61.3\% accuracy (against a random baseline of 33.3\%), while our system achieved 93.3\%.

\subsection{E3: SimHash Grounding Fidelity}
To prove that our hypervectors maintain the properties of the original transformer space, we measured the correlation between cosine similarities in $\mathbb{R}^{384}$ and hypervector space. Over 30 seeds, the Pearson correlation was $0.9921 \pm 0.0002$ and Spearman correlation was $0.9907 \pm 0.0002$, confirming exceptional semantic fidelity.

\section{Discussion and Limitations}
\label{sec:discussion}

While Astrum Verum significantly improves structural recall, several limitations remain. 

\textbf{Extraction Fidelity:} The system depends on the LLM's ability to accurately extract triplets. Errors in the parsing phase (e.g., misidentifying the subject) propagate directly into memory. Future iterations should incorporate uncertainty estimates from the LLM during the SimHash grounding phase.

\textbf{Multiple Answers and Collisions:} When a query has multiple valid answers (e.g., "Who does Alice like?" where she likes both Bob and Charlie), the current Match Quality metric returns only the top candidate. Extending the system to return a ranked list of candidates by relaxing the early exit strategy is a necessary step for open-world applications.

\textbf{Fixed Dimensionality:} We utilized $D=10,000$ based on classical HDC benchmarks. While this provides a high noise floor, it introduces significant computational overhead compared to flat RAG. Future work will investigate the trade-offs between dimensionality, capacity, and retrieval speed, particularly utilizing hardware acceleration \cite{kleyko2022vector}.

\section{Conclusion}
By combining transformer embeddings
 with hyperdimensional algebra, Astrum Verum achieves structural cognitive memory without the combinatorial explosion of graph databases or the hallucinations of continuous RAG. 

The introduction of the Match Quality metric solves the problem of decoded clarity hallucinations, ensuring that the system returns strict boolean states ($\emptyset$ or a grounded concept) rather than forced approximations. Furthermore, utilizing an Item Memory array bypasses the traditional superposition limits of VSA. 

\textbf{Integration and Validation:} To evaluate the practical utility of this architecture, we have initiated the integration and performance testing of Astrum Verum as a specialized cognitive skill within the \textbf{Google Antigravity autonomous agent framework}. This phase aims to validate the effectiveness of structural hypervectors for verifiable fact-recall and episodic tracking in real-world autonomous orchestrations. Future work will involve further optimization of the symbolic extraction pipeline and exploring hardware-accelerated VSA operations \cite{kleyko2022vector}.
